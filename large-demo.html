Large Countdown Timer Demo

This page demonstrates the new large mode with 150px high digits and flip animations, perfect for center page display in Moodle.

Large Mode Preview

Features

  • 150px High Digits: Massive, highly visible countdown numbers
  • Flip Animations: Smooth rolodex-style flip transitions when numbers change
  • Full Width: Takes up 100% of the container width
  • Airport Board Style: Premium look with gradients and shadows
  • Large Colons: Visible separators between time units
  • Glowing Labels: Enhanced typography with text shadows
