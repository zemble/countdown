Recommended Moodle iframe Code

<div style="text-align: center; margin: 10px 0;">
    <iframe src="https://zemble.github.io/countdown/?date=2025-10-10&time=17:00:00&timezone=Pacific/Auckland&title=Assignment%20Due&compact=true" width="100%" height="220" style="border:none; max-width: 100%;"></iframe>
</div>

Key changes for Moodle:

  • Use width="100%" instead of fixed pixels
  • Add max-width: 100%; to prevent overflow
  • Use height="220" to fit the countdown properly
  • Keep the compact=true parameter
